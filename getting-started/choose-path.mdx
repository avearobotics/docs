## Connect the cameras

<Columns cols={2}>
  <Card title="Supported camera" icon="camera" href="/getting-started/supported-hardware#cameras">
    Let Sentinel open and manage a supported camera attached to the robot computer.
  </Card>
  <Card title="ROS 2 camera" icon="video" href="/ros2/camera-interface">
    Publish a camera feed from an existing ROS 2 system.
  </Card>
</Columns>

<Note>
  You can mix these paths. A supported robot can use a custom ROS 2 camera, and a ROS 2 robot can use a supported camera that Sentinel opens directly.
</Note>

Robot configuration is moving into website-managed form fields. During the transition, some integrations may still use a generated `robot.yaml`.
