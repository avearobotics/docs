These are known-good starting configs for common setups. We provide the actual `.yaml` files — ask us on [Slack](https://avea-robotics.slack.com) for the ones that match your hardware.

<Note>
  The reference `.yaml` files referenced below are distributed by Avea (they were attached in our internal docs). They aren't checked into this site yet — request the file for your setup and we'll send it.
</Note>

## Bimanual YAM

- 2× RealSense D405 wrist cameras and 1× D405 overhead (easily swapped for D455 or D435i)
- Works out of the box with the `MolmoAct2-Bimanual-Yam` checkpoint configuration

**Reference file:** `yam_dual_3realsense.yaml`

## Bimanual YAM with immersive neck

- 2× RealSense D405 wrist cameras and 1× ZED 2 Mini
- Uses `NVENC`, available only on `amd64-cu13` containers

**Reference file:** `yam_dual_zed_immersive.yaml`

## Bimanual Trossen WXAI

**Reference file:** `trossen_dual_3realsense.yaml`

## Bimanual Trossen WXAI with immersive neck
